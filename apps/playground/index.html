Open Source Email Editor

🕊 Lit Pigeon

Open-source drag & drop email editor built with web components

Try the Playground GitHub
$ npm install @lit-pigeon/editor

Features

Everything you need to build beautiful email templates

Drag & Drop

Intuitive visual editor with drag and drop block placement, reordering, and multi-column layouts.

MJML Export

Render designs to MJML and responsive HTML email that works across all major email clients.

MJML Import

Import existing MJML templates directly into the editor for further editing and customization.

Framework Agnostic

Built on web components with Lit. Works with React, Angular, Vue, or vanilla JavaScript.

Asset Manager

Built-in asset management for images and files with upload and library browsing support.

Merge Tags

Dynamic content placeholders for personalization — names, links, and custom data fields.

Packages

Modular architecture — use only what you need

@lit-pigeon/core Document model, block types, and utilities
@lit-pigeon/editor Drag & drop visual editor web component
@lit-pigeon/renderer-mjml Render documents to MJML and HTML
@lit-pigeon/parser-mjml Import MJML into the editor format
@lit-pigeon/angular Angular wrapper for the editor component

Quick Start

Get up and running in seconds

import '@lit-pigeon/editor';
import { createDefaultDocument } from '@lit-pigeon/core';

const editor = document.querySelector('pigeon-editor');
editor.document = createDefaultDocument('My Email');
