# AI-spec changelog

Spec versions are independent from the npm package versions. Breaking
document-shape changes bump the major (`2.0`); additive changes bump the
minor (`1.1`).

## 1.0

- Initial published spec: `PigeonDocument` version `"1.0"` — body attributes,
  rows/columns with 12-unit ratios, the typed block catalog
  (text/image/button/divider/spacer/social/html/hero/navbar), merge-tag
  conventions, and the JSON Schema
  ([`lit-pigeon-ai-spec.schema.json`](./lit-pigeon-ai-spec.schema.json)).
